# Rating

The `Rating` component is used to display ratings using stars or custom indicators.

```js
import { Rating } from '@sajari/react-components';
```

## Usage

```jsx
<Rating value={3} />
```

### Maxium stars

```jsx
<Rating max={10} value={5} />
```

### Half star

```jsx
<div style={{ display: 'flex', flexDirection: 'column' }}>
  <Rating value={2.9} />
  <Rating value={3.5} />
  <Rating value={4.1} />
  <Rating value={5} />
</div>
```


### Copywriting

```jsx
<div style={{ display: 'flex', flexDirection: 'column' }}>
  <Rating value={1} character="A">
    <span style={{ marginLeft: '1rem' }}>A for Amazing</span>
  </Rating>
  <Rating value={2.5} character="B">
    <span style={{ marginLeft: '1rem' }}>B for Brilliant</span>
  </Rating>
  <Rating value={5} character="S">
    <span style={{ marginLeft: '1rem' }}>S for Search.io</span>
  </Rating>
</div>
```

### Direction

```jsx
<div style={{ display: 'flex', flexDirection: 'column' }}>
  <Rating direction="rtl" value={1.5} character="A">
    <span style={{ marginRight: '1rem' }}>A for Amazing</span>
  </Rating>
  <Rating value={2.5} character="B">
    <span style={{ marginLeft: '1rem' }}>B for Brilliant</span>
  </Rating>
  <Rating direction="rtl" value={5} character="S">
    <span style={{ marginRight: '1rem' }}>S for Search.io</span>
  </Rating>
</div>
```

## Accessibility

- Container has `role` img and corresponding label e.g "Rating: 2 out of 5 stars".
- Children has `hidden` true
- `aria-setsize` = `count` and `aria-posinset` = `index + 1`

## Props

| Name        | Type                                            | Default   | Description                                             |
| ----------- | ----------------------------------------------- | --------- | ------------------------------------------------------- |
| `value`     | `number`                                        |           | The value of this rating                                |
| `max`       | `number`                                        | 5         | The maximum rating.                                     |
| `children`  | `React.ReactNode`                               | `null`    | Children for copywriting.                               |
| `character` | `React.ReactNode` \| `React.FC<{index, count}>` | `<Star/>` | The component used to render rating.                    |
| `direction` | `'ltr'` \| `'rtl'`                              | `'ltr'`   |                                                         |
| `unit`      | `string`                                        | `'star'`  | Singular form, used in label to convey more accurately. |

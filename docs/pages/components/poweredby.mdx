# PoweredBy

The `PoweredBy` component is used to display that you're awesome and Sajari powering your search UI ðŸ’ª. If you're on a starter plan, it is required to display this component where Sajari search logic is used.

```js
import { PoweredBy } from '@sajari/react-search-ui';
```

## Usage

```jsx
<div className="space-x-6">
  <PoweredBy />
  <PoweredBy appearance="color" />
</div>
```

## Props

| Name         | Type            | Default | Description                       |
| ------------ | --------------- | ------- | --------------------------------- |
| `appearance` | `mono`, `color` | `mono`  | The apperance of the Sajari logo. |

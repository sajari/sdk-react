# PoweredBy

The `PoweredBy` component is used to display that you're awesome and Search.io is powering your search UI ðŸ’ª. If you're on a starter plan, it is required to display this component where Search.io search logic is used.

```js
import { PoweredBy } from '@sajari/react-search-ui';
```

## Usage

```jsx
<div className="space-x-6">
  <PoweredBy />
  <PoweredBy appearance="color" />
</div>
```

## Props

| Name         | Type            | Default | Description                           |
| ------------ | --------------- | ------- | ------------------------------------- |
| `appearance` | `mono`, `color` | `mono`  | The appearance of the Search.io logo. |

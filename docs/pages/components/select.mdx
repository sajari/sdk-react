# Select

Select component is a component that allows users pick a value from predefined options. Ideally, it should be used when there are more than 5 options, otherwise you might consider using a radio group instead.

```js
import { Select } from '@sajari/react-components';
```

## Usage

Here's a basic usage of the Select component.

```jsx
<Select placeholder="Select an option">
  <option value="option1">Option 1</option>
  <option value="option2">Option 2</option>
  <option value="option3">Option 3</option>
</Select>
```

### Invalid

```jsx
<Select placeholder="Select an option" invalid>
  <option value="option1">Option 1</option>
  <option value="option2">Option 2</option>
  <option value="option3">Option 3</option>
</Select>
```

## Disabled

```jsx
<Select placeholder="Select an option" disabled>
  <option value="option1">Option 1</option>
  <option value="option2">Option 2</option>
  <option value="option3">Option 3</option>
</Select>
```

## Props

The Select composes the `Box` component so you can pass props for
`Box`. These are props related to the Select component.

| Name      | Type      | Default | Description                                                   |
| --------- | --------- | ------- | ------------------------------------------------------------- |
| `invalid` | `boolean` |         | Whether the select is invalid                                 |
| `label`   | `string`  |         | An aria-label, also used for the placeholder if not specified |

# Text

The `Text` component is used to display text.

## Import

```js
import { Text } from '@sajari/react-components';
```

## Usage

```jsx
<div className="flex flex-col space-y-2">
  <Text>Default</Text>
  <Text as="small">Small</Text>
  <Text as="code">Code</Text>
  <Text as="mark">Mark</Text>
</div>
```

## Props

| Name       | Type                  | Default | Description                                                                                                                                                                                                                                                                              |
| ---------- | --------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `as`       | `string`              | `"p"`   | One of `"small"` \| `"p"` \| `"h1"` \| `"h2"` \| `"h3"` \| `"h4"` \| `"h5"` \| `"h6"` \| `"figcaption"` \| `"pre"` \| `"code"` \| `"code"` \| `"em"` \| `"i"` \| `"u"` \| `"cite"` \| `"del"` \| `"del"` \| `"ins"` \| `"mark"` \| `"s"` \| `"samp"` \| `"sub"` \| `"sup"` \| `"strong"` |
| `truncate` | `boolean` \| `number` | false   | `true` is equal to truncating 1 line and so forth                                                                                                                                                                                                                                        |

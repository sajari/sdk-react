Core components that allow you to build a beautiful search interface with any back end.

## 1. Install

Install the dependency via `yarn` or `npm install`.

```txt
yarn add @sajari/react-components
```

## 2. Import

Pick and choose the components you'd like to use and import them.

```js
import { Button } from '@sajari/react-components';
```

## 3. Use

Use them in your interface.

```jsx
<Button appearance="primary">Yay!</Button>
```

# Swatch

The `Swatch` component acts like a color filter that allows for more granular product/item searching.

```js
import { Swatch } from '@sajari/react-components';
```

## Usage

```jsx
function BasicUsage() {
  const [checkedColors, setCheckedColors] = React.useState([]);
  return (
    <Swatch checkedColors={checkedColors} onChange={setCheckedColors}>
      <Swatch.Color.White />
      <Swatch.Color.Silver />
      <Swatch.Color.Black />
      <Swatch.Color.Pink />
      <Swatch.Color.Magenta />
      <Swatch.Color.Red />
      <Swatch.Color.Beige />
      <Swatch.Color.Orange />
      <Swatch.Color.Brown />
      <Swatch.Color.Yellow />
      <Swatch.Color.Green />
      <Swatch.Color.Azure />
      <Swatch.Color.Aqua />
      <Swatch.Color.Teal />
      <Swatch.Color.Turquoise />
      <Swatch.Color.Blue />
      <Swatch.Color.ElectricBlue />
      <Swatch.Color.Lilac />
      <Swatch.Color.Purple />
      <Swatch.Color.Violet />
    </Swatch>
  );
}
```

### Colors

You can use the pre-defined set of colors by using the syntax `<Swatch.Color.{Color} />`, or you can use your own using `<Swatch.Color />` and pass the required props. Accepted format includes `hex`, `rgb`, `rgba`, `hsl`, `hsla`.

```jsx
function CustomColor() {
  const [checkedColors, setCheckedColors] = React.useState([]);
  return (
    <Swatch checkedColors={checkedColors} onChange={setCheckedColors}>
      <Swatch.Color id="hex" bg="#f56565" />
      <Swatch.Color id="rgb" bg="rgb(45, 100, 219)" />
      <Swatch.Color id="rgba" bg="rgba(45, 100, 219, 0.5)" />
      <Swatch.Color id="hsl" bg="hsl(309, 35%, 69%)" />
      <Swatch.Color id="css" bg="blue" />
    </Swatch>
  );
}
```

## Props

### Swatch

| Name            | Type                   | Default    | Description                                |
| --------------- | ---------------------- | ---------- | ------------------------------------------ |
| `checkedColors` | `string[]`             | `[]`       | An array containing id of selected colors. |
| `onChange`      | `(id: string) => void` | `() => {}` | The handler.                               |

### Swatch.Color

| Name     | Type     | Default | Description                                                                                                                                         |
| -------- | -------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `id`     | `string` |         | The id of the color.                                                                                                                                |
| `bg`     | `string` |         | The background color. See above for accepted formats.                                                                                               |
| `color`  | `string` |         | The color of the checkmark when that color is selected. Defaults to being calculated based on the background color. See above for accepted formats. |
| `border` | `string` |         | The border color. Defaults to using `bg` darkened by 10% if not specified. See above for accepted formats.                                          |

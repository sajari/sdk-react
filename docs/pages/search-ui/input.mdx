# Input

Input component is a component that is used to get user input in a text field. It is usually used together with a FormControl to provide an accessible label, validation messages, etc.

## Import

```js
import { Input } from 'sajari/react-search-ui';
```

## Usage

```jsx
<>
  <Input label="Search something" />
  <Results className="mt-4" />
</>
```

## Standard

```jsx
<>
  <Input label="Search something" mode="standard" />
  <Results className="mt-4" />
</>
```

## Suggestions

```jsx
<>
  <Input label="Search something" mode="suggestions" />
  <Results className="mt-4" />
</>
```

## Results

```jsx
<Input label="Search something" mode="results" />
```

## Typeahead

```jsx
<>
  <Input label="Search something" mode="typeahead" />
  <Results className="mt-4" />
</>
```

## Standard

By default `<Input />` has an additional mode `"instant"` and it is also the default value (search on change), if you want to disable it, simple change mode to `"standard"`:

```jsx
function Example() {
  const [value, setValue] = React.useState('');

  return (
    <form
      onSubmit={(event) => {
        event.preventDefault();
        const formData = new FormData(event.target);
        const value = formData.get('query');
        setValue(value);
      }}
      className="flex flex-col space-y-3"
    >
      <Input mode="standard" name="query" label="Search something" />
      <p>You submited: {value}</p>
    </form>
  );
}
```

## Props

`Input` component props inherits the `input` element props with some additional for speech recognition and loading state

| Name       | Type                                                            | Default      | Description                                                                                                                                                                                                                                                                                               |
| ---------- | --------------------------------------------------------------- | ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `mode`     | `'standard'` \| `'typeahead'` \| `'suggestions'` \| `'results'` | `'standard'` | The mode of the combobox. `'standard'` will operate as a normal text input. `'typeahead`' will provide an inline completion suggestion. `'suggestions'` will provide an autocomplete menu with options to select. `'results'` will perform a full search using the provided term and render full results. |
| `onChange` | `(value: string) => void`                                       |              | The callback when the input value changes. Note: The `event` is not passed as the argument.                                                                                                                                                                                                               |

# Input

Input component is a component that is used to get user input in a text field. It is usually used together with a FormControl to provide an accessible label, validation messages, etc.

## Import

```js
import { Input } from 'sajari/react-search-ui';
```

## Results

```jsx
<Input label="Search something" mode="results" />
```

## Props

`Input` component props inherits the `input` element props with some additional for speech recognition and loading state

| Name           | Type                  | Default | Description                                                                                                                      |
| -------------- | --------------------- | ------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `loading`      | `boolean`             | `false` | Show the spinner icon                                                                                                            |
| `enableVoice`  | `boolean`             | `false` | Whether to enable speech recognition ([Compatibility table](https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition)) |
| `onVoiceInput` | `(r: string) => void` |         | The callback when a voice transcript is output                                                                                   | --> |

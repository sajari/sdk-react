# Pagination

The Pagination component renders a pagination component with config wired in.

```js
import { Pagination, Results } from '@sajari/react-search-ui';
import { SearchProvider, useSearch } from '@sajari/react-hooks';
```

## Usage

```jsx
function Example() {
  const pipeline = new Pipeline(
    {
      account: '1594153711901724220',
      collection: 'bestbuy',
      endpoint: 'https://jsonapi-us-valkyrie.sajari.net',
    },
    'query',
  );

  const variables = new Variables({ resultsPerPage: 3 });

  const SearchPlayground = React.memo(() => {
    useSearch('iphone');

    return (
      <div className="space-y-6">
        <Results appearance="grid" />
        <Pagination />
      </div>
    );
  });

  return (
    <SearchProvider search={{ pipeline, variables, fields: { title: 'name', subtitle: 'brand' } }}>
      <SearchPlayground />
    </SearchProvider>
  );
}
```

## Props

| Name    | Type                                | Default    | Description                  |
| ------- | ----------------------------------- | ---------- | ---------------------------- |
| `align` | `'left'` \| `'center'` \| `'right'` | `'center'` | How to align the pagination. |

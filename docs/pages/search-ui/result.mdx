# Result

Result is a component used to display a single search result.

```js
import { Result } from '@sajari/react-search-ui';
```

## Usage

```jsx
<Result
  image="https://source.unsplash.com/random/100x100"
  title="Apple - iPhone 6s Plus 64GB - Space Gray"
  rating={9}
  ratingMax={10}
  description="A 5.5-inch Retina HD display with 3D Touch. 7000 series aluminum and stronger cover glass. An A9 chip with64-bit desktop-class architecture. All new 12MP iSight camera with Live Photos. Touch ID. Faster LTE and Wi-Fi. Long battery life and iOS 9 and iCloud. All in a smooth, continuous unibody design."
  category="Cell Phones > iPhone"
/>
```

## Props

| Name          | Type                | Default | Description                                        |
| ------------- | ------------------- | ------- | -------------------------------------------------- |
| `appearance`  | `"row"` \| `"grid"` | `"row"` | The layout for displaying the results.             |
| `url`         | `string`            |         | The url to redirect to when the result is clicked. |
| `title`       | `string`            |         |                                                    |
| `image`       | `string`            |         | The image's link.                                  |
| `description` | `string`            |         |                                                    |
| `category`    | `string`            |         |                                                    |
| `rating`      | `number`            |         |                                                    |
| `ratingMax`   | `number`            | `5`     |                                                    |

# Result

Result is a component used to display a single search result.

```js
import { Result } from '@sajari/react-search-ui';
```

## Usage

```jsx
<Result />
```

## Props

| Name         | Type                      | Default | Description                       |
| ------------ | ------------------------- | ------- | --------------------------------- |
| `align`      | `left`, `center`, `right` | `right` | The alignment of the content.     |
| `appearance` | `mono`, `color`           | `mono`  | The apperance of the Sajari logo. |

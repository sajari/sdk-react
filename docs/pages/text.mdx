# Text

The Text component is used to display text.

## Import

```js
import { Text } from 'sajari/react-components';
```

## Usage

```jsx
<>
  <Text>Default</Text>
  <Text as="small">Small</Text>
  <Text as="code">Code</Text>
  <Text as="mark">Mark</Text>
</>
```

The `<Swatch />` component

### Colors

You can use the pre-defined set of colors by using the syntax `<Color.{Color} />`, or you can use your own using `<Swatch.Color />` and pass the required props.

<Playground>
  {() => {
    const [checkedColors, setCheckedColors] = React.useState([]);
    return (
      <ExampleContainer style={{display: "flex"}}>
        <Swatch checkedColors={checkedColors} onChange={setCheckedColors}>
          <Swatch.Color id="Magenta" bg="#F56565" color="#ECC94B" border="#4299E1" />
        </Swatch>
      </ExampleContainer>
    );
  }}
</Playground>

### Overriding prop

If there are two duplicating ids in the Swatch, simply override the props.

<Playground>
  {() => {
    const [checkedColors, setCheckedColors] = React.useState([]);
    return (
      <ExampleContainer style={{display: "flex"}}>
        <Swatch checkedColors={checkedColors} onChange={setCheckedColors}>
          <Swatch.Color id="Magenta" bg="#F56565" color="#ECC94B" border="#4299E1" />
          <Color.Magenta id="Magenta (override)" />
        </Swatch>
      </ExampleContainer>
    );
  }}
</Playground>

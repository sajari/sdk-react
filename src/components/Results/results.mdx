# Results

`<Results />` renders the list of results using the component passed to the `ResultRenderer` prop, defaulting
to a standard `Result` component if nothing is provided.

```javascript
// Import the Results component
import { Results } from "@sajari/sdk-react"
```

## Usage

### Basic
<Playground>
    <Results />
</Playground>

### With images
<Playground>
    <Results showImages />
</Playground>


### With custom styles
<Playground>
    <Results styles={{
        container: {
            flexWrap: "nowrap",
            justifyContent: "space-between"
        },
        item: {
            width: "32%",
            padding: "0.5rem",
            boxShadow: "0 2px 2px 0 rgba(0,0,0,0.16), 0 0 0 1px rgba(0,0,0,0.08)",
            marginBottom: 0
        },
        result: {
            container: {
                height: "100%",
                justifyContent: "space-between"
            }
        }
    }} />
</Playground>

### With custom result renderer
<ExampleContainer>
    <Results
        ResultRenderer={({values, token}) => <a href={token || "#"}>{values.title}</a>}
        styles={{
            item: {
                marginBottom: "0.25rem",
                "&:last-of-type": {
                    marginBottom: 0
                }
            }
        }}
    />
</ExampleContainer>

```javascript
import { Results } from "@sajari/sdk-react"

const CustomRenderer = ({values, token}) => <a href={token || "#"}>{values.title}</a>

<Results
    ResultRenderer={CustomRenderer}
    styles={{
        item: {
            marginBottom: "0.25rem"
            "&:last-of-type": {
                marginBottom: 0
            }
        }
    }}
/>
```

# Select

## Usage

<ExampleContainer>
  <ThemeProvider>
    <Select placeholder="Select an option">
      <option value="option1">Option 1</option>
      <option value="option2">Option 2</option>
      <option value="option3">Option 3</option>
    </Select>
  </ThemeProvider>

## With Filter

Normally, `Select` acts like a normal `select` tag, but you can use it with a Filter

<Playground>
  {() => {
    const filter = categoryFilter();
    const [output, setOutput] = React.useState(filter.filter());


    filter.listen(EVENT_SELECTION_UPDATED, () => {
      setOutput(filter.filter())
    })

    return (
      <ExampleContainer>
        <ThemeProvider>
          <FilterProvider filter={filter}>
            <FilterConsumer>
              {({options, selected, set}) => {

                return (
                  <Select onChange={e => set(e.target.value, true)}>
                    {Object.entries(options).map(([key, value], index) => <option value={key} key={`option-${key}-${value}-${index}`}>{key}</option>)}
                  </Select>
                )
              }}
            </FilterConsumer>
          </FilterProvider>
          <pre>Output: {output}</pre>
        </ThemeProvider>
      </ExampleContainer>
    )

}}

</Playground>


## Migration guide (v3)

If you use the `Select` component prior to `v3` it would look something like this:

```javascript
import { Select } from '@sajari/sdk-react';

<Select filter={categories} />;
```

The reason this works is because under the hood we use the same thing (FilterProvider, FilterConsumer, ...) all wraped inside `Select`, but we've decided from `v3` to use plain `select` tag and instead shift the control to you therefore giving you maximum controllability.

```javascript
import { Select, FilterProvider, FilterConsumer } from '@sajari/sdk-react';

<FilterProvider filter={filter}>
  <FilterConsumer>
    {' '}
    // useContext is also an option here
    {({ options, selected, set }) => {
      return <Select>...</Select>;
    }}
  </FilterConsumer>
</FilterProvider>;
```

# Input

The `Input` component provides a text-box which performs searches as the user types.
It is customizable through the use of props, options include rendering autocomplete
suggestions directly in the input box, or alternatively in
a drop-down list underneath.

```javascript
// Import the Input component
import { Input } from "@sajari/sdk-react";
```

## Usage

### With default props

With the `default` props, the `<Input />` acts like a regular `<input type="text" />` html element.

<Playground>
  <InputWrapper>
    <Input />
  </InputWrapper>
</Playground>

### With default value

<Playground>
  <InputWrapper>
    <Input defaultValue="A default string value" />
  </InputWrapper>
</Playground>

### With a clear button

This example uses `showClearButton` property. Clicking this button will reset the input back to its default state.

<Playground>
  <InputWrapper>
    <Input showClearButton />
  </InputWrapper>
</Playground>

### With instant search and typeahead

<Playground>
  <InputWrapper>
    <Input mode="typeahead" instantSearch showClearButton />
  </InputWrapper>
</Playground>

### With suggestions dropdown

<Playground>
  <InputWrapper style={{ height: 300 }}>
    <Input mode="typeahead" dropdownMode="suggestions" showClearButton />
  </InputWrapper>
</Playground>

### With results dropdown

<Playground>
  <InputWrapper style={{ height: 400 }}>
    <Input
      mode="typeahead"
      dropdownMode="results"
      instantSearch
      showClearButton
    />
  </InputWrapper>
</Playground>

### With custom styles

<Playground>
  <InputWrapper style={{ height: 300 }}>
    <Input
      showClearButton
      mode="typeahead"
      dropdownMode="suggestions"
      styles={{
        container: {
          width: 300
        },
        input: isFocused => ({
          boxShadow: "none",
          borderRadius: 0,
          "&:hover": {
            boxShadow: "none"
          },
          borderBottom: isFocused ? "2px solid #3B5998" : "1px solid #737373",
          outline: 0
        }),
        button: {
          "&:hover": {
            color: "#3B5998"
          }
        },
        suggestions: {
          item: isFocused => ({
            color: isFocused ? "#fff" : "inherit",
            backgroundColor: isFocused ? "#3B5998" : "inherit"
          })
        }
      }}
    />
  </InputWrapper>
</Playground>

## Styles

### `css` classes

```css
/* input */
.sj-input {
}
.sj-input__input {
}
.sj-input__input.sj-input__input--focused {
}

/* button */
.sj-input__button {
}

/* typeahead */
.sj-input__typeahead {
}

/* suggestions */
.sj-input__suggestions {
}
.sj-input__suggestions__item

}
```

### `styles` prop

```typescript
type InputStyles = {
  container?: CSSObject;
  input?: (isFocused: boolean) => CSSObject;
  typeahead?: CSSObject;
  button?: CSSObject;
  suggestions?: {
    container?: CSSObject;
    item?: (isFocused: boolean) => CSSObject;
  };
};
```

# Input

The `Input` component provides a text-box which performs searches as the user types.
It is customizable through the use of props, options include rendering autocomplete
suggestions directly in the input box, or alternatively in
a drop-down list underneath.

```javascript
// Import the Input component
import { Input } from "@sajari/sdk-react"
```

## Usage

### With default props
With the `default` props, the `<Input />` acts like a regular `<input type="text" />` html element.
<Playground>
    <InputWrapper>
        <Input />
    </InputWrapper>
</Playground>

### With instant search and typeahead
<Playground>
    <InputWrapper>
        <Input mode="typeahead" instantSearch />
    </InputWrapper>
</Playground>

### With suggestions dropdown
<Playground>
    <InputWrapper style={{height: 300}}>
        <Input mode="typeahead" dropdownMode="suggestions" />
    </InputWrapper>
</Playground>

### With results dropdown
<Playground>
    <InputWrapper style={{height: 300}}>
        <Input mode="typeahead" dropdownMode="results" instantSearch />
    </InputWrapper>
</Playground>



### With custom styles
<Playground>
    <InputWrapper style={{height: 300}}>
        <Input mode="typeahead" dropdownMode="suggestions" styles={{
            container: () => ({
                backgroundColor: "#999",
                color: "#fff",
                border: "1px solid #999",
                boxShadow: "none"
            }),
            input: {
                "input::placeholder": {
                    color: "#eee"
                }
            },
            button: {
                color: "#eee"
            },
            suggestions: {
                container: {
                    backgroundColor: "#999",
                    color: "#eee"
                },
                item: (isHighlighted) => ({
                    color: isHighlighted ? "#222" : "inherit"
                })
            }
        }} />
    </InputWrapper>
</Playground>

# Search

Although the sdk provides its own `Input` component, a pre-built component like the `Input` will not be flexible enought to cover all cases when building search interfaces. The `Search` component, thus, provides a solution to address the issue by using the render prop technique.

```javascript
// Import the Search component
import { Search } from "@sajari/sdk-react";
```

The component was built on top of [downshift](https://github.com/paypal/downshift), providing all of the library funtionalities along with `pipeline` methods & responses.

#### Pipleline methods & responses

- `suggestions`: `string[]`
- `results`: `Result[]`
- `completion`: `string`
- `paginate`: `(page: number) => void`
- `search`: `(query: string, override: boolean) => void`
- `instantSearch`: `(query: string, override: boolean) => void`

They are available to use when call:

- Render function: `<Search>{searchProps => <div>/* your JSX here! */</div>}</Search>`
- Props: `stateReduder` or `onChange`.

## Usage

### Add custom clear button

<Playground>
  <InputWrapper style={{ height: 250 }}>
    <Search>
      {({
        getInputProps,
        inputValue,
        isOpen,
        getMenuProps,
        results,
        reset,
        search
      }) => (

          <div
            style={{
              position: "relative"
            }}
          >
            <input

              {...getInputProps({

                  search(v.target.value);
                }
              })}
              style={{
                borderRadius: "4px",
                border: "1px solid #d9d9d9",
                width: "100%",
                height: "42px",
                display: "block",
                boxShadow: "0 2px 3px rgba(0,0,0,0.05)",
                padding: "5px 50px 5px 10px",
                outline: "none",
                fontSize: "20px"
              }}
            />
            {inputValue && (
              <button
                style={{
                  border: "none",
                  position: "absolute",
                  background: "transparent",
                  right: "0",
                  top: "0",
                  height: "42px",
                  padding: "4px 10px 0 10px",
                  cursor: "pointer",
                  fontSize: "22px",
                  color: "#777777"
                }}
                onClick={() => {
                  reset();
                }}
              >
                <TimesIcon />
              </button>
            )}
          </div>
          {isOpen && (
            <ol style={{ margin: "20px 0 0 0", padding: "0" }}>
              {results.map(({ values: { url, title } }, index) => (
                <li key={index}>
                  <a href={url} target="__blank" rel="noopener">
                    {title}
                  </a>
                </li>
              ))}
            </ol>
          )}
        </div>
      )}
    </Search>
  </InputWrapper>
</Playground>

### Search when hit enter or click on button

<Playground>
  <InputWrapper style={{ height: 250 }}>
    <Search>
      {({ getInputProps, inputValue, search, results }) => (
        <div style={{ margin: "auto", maxWidth: "500px" }}>
          <div style={{ position: "relative", marginBottom: "20px" }}>
            <input
              placeholder="Type and hit enter..."
              type="text"
              {...getInputProps({
                onKeyDown: e => {
                  if (e.key === "Enter" && inputValue) {
                    search(inputValue, false);
                  }
                }
              })}
              style={{
                borderRadius: "4px",
                border: "1px solid #d9d9d9",
                width: "100%",
                height: "42px",
                display: "block",
                boxShadow: "0 2px 3px rgba(0,0,0,0.05)",
                padding: "5px 70px 5px 10px",
                outline: "none",
                fontSize: "18px"
              }}
            />
            <button
              style={{
                position: "absolute",
                right: "0",
                top: "0",
                height: "42px",
                padding: "4px 10px 0 10px",
                border: "none",
                backgroundColor: "transparent",
                fontSize: "24px",
                color: "#777777",
                cursor: "pointer"
              }}
              onClick={e => {
                e.preventDefault();
                if (inputValue) {
                  search(inputValue, false);
                }
              }}
            >
              <SearchIcon />
            </button>
          </div>
          {results.length > 0 && (
            <ol style={{ margin: "0", padding: "0" }}>
              {results.map(({ values: { url, title } }, index) => (
                <li key={index}>
                  <a href={url} target="__blank" rel="noopener">
                    {title}
                  </a>
                </li>
              ))}
            </ol>
          )}
        </div>
      )}
    </Search>
  </InputWrapper>
</Playground>

### Search with suggestions

<Playground>
  <InputWrapper style={{ height: 250 }}>
    <Search
      itemToString={item => item || ''}
      stateReducer={(state, changes, { search }) => {
        const { type } = changes;
        if (type === Search.stateChangeTypes.keyDownEnter) {
          search(changes.inputValue, false);
        }
        return changes;
      }}
    >
      {({
        getInputProps,
        inputValue,
        isOpen,
        getMenuProps,
        suggestions,
        highlightedIndex,
        getItemProps,
        selectedItem,
        reset,
        instantSearch,
        results,
      }) => {
        const isShow = isOpen && suggestions.length > 0;
        return (
          <div style={{ margin: "auto", maxWidth: "500px" }}>
            <div style={{ textAlign: 'center' }}>
              <input
              placeholder="Type to search..."
                type="text"
                {...getInputProps({
                  onChange: v => {
                    instantSearch(v.target.value);
                  },
                })}
                style={{
                  borderRadius: isShow ? '4px 4px 0 0' : '4px',
                  border: '1px solid #d9d9d9',
                  width: '100%',
                  height: '42px',
                  display: 'block',
                  boxShadow: '0 2px 3px rgba(0,0,0,0.05)',
                  padding: '5px 10px',
                  outline: 'none',
                  fontSize: '18px',
                }}
              />
            </div>
            {isShow && (
              <ul
                {...getMenuProps()}
                style={{
                  boxShadow: '0 2px 3px rgba(0,0,0,0.05)',
                  listStyle: 'none',
                  border: '1px solid #d9d9d9',
                  padding: '0',
                  margin: '-1px 0 0 0',
                  borderRadius: '0 0 4px 4px',
                  overflow: 'hidden',
                }}
              >
                {suggestions.map((item, index) => (
                  <li
                    highlighted={highlightedIndex === index}
                    selected={selectedItem === item}
                    style={{
                      fontSize: '18px',
                      padding: '5px 10px',
                      backgroundColor:
                        highlightedIndex === index
                          ? '#f6f6f6'
                          : selectedItem === item
                          ? '#eeeeee'
                          : '#ffffff',
                    }}
                    {...getItemProps({
                      key: item,
                      index,
                      item,
                    })}
                  >
                    {item}
                  </li>
                ))}
              </ul>
            )}
          </div>
        );
      }}
    </Search>

  </InputWrapper>
</Playground>
